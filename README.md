# PHY420\_W26\_Chua\_Circuit

This is a repository of programs created for the PHY420W Capstone project "Chua's Circuit: Construction and Modeling." It includes code that models the behavior of individual and coupled Chua circuits, and code used to process data from a realized Chua circuit.





### Modeling Programs



The programs in the "Modeling Programs" folder are those that numerically solve the governing equations of motion for a Chua circuit (or system of coupled circuits) and output the resulting data in plot form.



The mathematical model on which this code is based is found in 

&#x20;   

Liu et al. "Synchronization control between two Chua's circuits via capacitive coupling." Applied Mathematics and Computing 360 (2019) pp. 96-104.



Additional formulae for dimensionless parameters are adapted from



Kennedy, "Robust Op-Amp realization of Chua's circuit" Frequenz vol 40 no. 3-4 (March-April 1992) pp. 66-80.





### Experimental Programs



The programs in the "Experimental Programs" folder are those that are used to process current and voltage data obtained from a realized Chua circuit.
